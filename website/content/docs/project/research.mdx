## Quick summary

The paper introduces a machine-verifiable trust layer for autonomous AI agents. Three primitives combined: Unified Agent Identity Tokens (UAIT), W3C Verifiable Credentials with Ed25519Signature2020 proofs, and a hash-chained audit trail with optional Base L2 testnet anchoring via the Ethereum Attestation Service.
